11.3.235 throw/1   [ISO]

Synopsis

throw(+Exception)

Exception is thrown as an exception.

Arguments

Exception
term, must be nonvar

Exceptions

instantiation_error
Exception is unbound.

See Also

ref-ere, on_exception/1.
