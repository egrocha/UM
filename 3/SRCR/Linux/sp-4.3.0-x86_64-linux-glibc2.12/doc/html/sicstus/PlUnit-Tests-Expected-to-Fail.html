10.46.3.3 Tests Expected to Fail

Tests that are expected to fail may be specified using the option fail or by negating the test-body using \+. 

     test(is_set) :-
             \+ is_set([a,a]).
     
     test(is_set, [fail]) :-
             is_set([a,a]).
