Resumo:

Este trabalho envolveu a transportação de dados armazenados em formato JSON para a base de dados GraphDB.

Como primeiro passo, os dados formatados em JSON foram transformados num ficheiro XML. Isto foi feito para que se possa usar uma stylesheet XSL para transformar estes dados em formato Turtle.

Foram também feitos ajustes manuais tanto ao ficheiro Turtle como ao ficheiro de dados XML para que os dados resultantes estivessem bem formatados.

Finalmente, os dados foram importados para uma instância da base de dados GraphDB, onde é possível se visualizar e consultar os dados.
