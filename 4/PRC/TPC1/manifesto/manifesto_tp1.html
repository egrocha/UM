Resumo:

Primeiro, foi criado um modelo para os dados processados através da ferramenta Protégé. De seguida, foi criada uma stylesheet XSL para converter os dados XML em formato Turtle.

Finalmente, os dados foram importados para uma instância da base de dados GraphDB, onde é possível se visualizar e consultar os dados.
