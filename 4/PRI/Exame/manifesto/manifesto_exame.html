Resumo:

Este exame envolveu a criação de duas aplicações: uma API para acesso a uma base de dados que contem informações relativa ao cinema, e outra que envolve a criação de uma interface para acesso e visualização de dados da plataforma CLAV.

Na criação da base de dados MongoDB para a aplicação Cinema foi utilizado um novo ficheiro moviesID.json que foi criado pelo script moviesAddID.js, para adicionar um novo ID a todos os registos.

Resultados:

  • Aplicação Cinema
  • Aplicação CLAV
