RESUMO

Este trabalho prático tem como objetivo o desenvolvimento de uma aplicação Web que gerencia um repositório digital de obras musicais e respetivas partituras. O repositório foi construído respeitando o modelo de referência internacional OAIS - (Open Archive Information System).

A aplicação proposta pretende proporcionar a uma banda filarmónica (ou utilizador independente) a visualização e navegação de obras e respetivas partituras.

As tecnologias utilizadas para o desenvolvimento da aplicação são Node.js, desenvolvido para aplicações escaláveis e para armazenamento MongoDB, que funciona como base de dados com escalabilidade e flexibilidade.

Para a execução do programa, é necessário descarregar a pasta principal - iBanda. Depois disso, numa consola, escrever o comando "npm install" para instalar todos os componentes da aplicação. Logo após a instalação, será necessária ligação da base de dados mongo através do comando "mongod". Posto isto, na pasta da aplicação, executar o comando "npm start" para começar a aplicação. Abrindo o browser, e conectando em http://localhost:2727/, temos acesso à página principal da aplicação.
